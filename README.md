  ## Released
  Utility tools that achieved their scope.  Updated periodically.

  - [stakeout](./stakeout) — AI-powered git change tracking
  - [pdf2text](./pdf2text) — PDF text extraction with GUI
  - [keywallet](./keywallet) — password manager with encryption
  - [stickers](./stickers) — floating desktop stickers
  - [zeroslate](./zeroslate) — forensic drive wiper
  - [dustpan](./dustpan) — system cleaner, no ads, memberships, or upsells
  - [walkietalkie](./walkietalkie) — local network chat + file transfer
  - [colorsnag](./colorsnag) — global color picker from screen
  - [iconforge](./iconforge) — drag-and-drop icon generation & conversion
  - [quicknote](./quicknote) — hotkey floating note, daily markdown 
  - [uninstallplus](./uninstallplus) — Windows deep-clean uninstaller
  - [campeek](./campeek) — quick webcam preview hotkey
  - [porthog](./porthog) — see what's using your ports and kill it
  - [handoff](./handoff) — alternative to Anthropic's /compact for session recap
  - [audiofy](./audiofy) — extract audio from video
  - [pixelruler](./pixelruler) — screen measurement tool
  - [ffmpeg-sitter](./ffmpeg-sitter) — one-click FFmpeg installer/uninstaller for Windows
  - [pintop](./pintop) — hotkey to pin windows on top
  - [micpanic](./micpanic) — F10 to mute mic system-wide
  - [emojoy](./emojoy) — hotkey emoji picker
  - [caffeine](./caffeine) — prevent PC sleep

  *Personal Tools - No customer-facing UI; little documentation*

  ## Operational
  Currently being developed, stable enough to test, expect bugs and utilitarian UI.

  - [Roundtable](./Roundtable) — playable fiction with persistent AI characters
  - [Ear](./Ear) — audio perception for LLMs
  - [claude-starship](./claude-starship) — A Claude simulation that doubles as a desktop command center
  - [research-hub](./research-hub) — An attempt to synthesize novel research autonomously
  - [the-house](./the-house) — A simulation of experience with customizable residents
  - [hearthline](./hearthline) — An alternative to Anthropic's official UI for chat
  - [math-mind](./math-mind) — Small transformer (~85M) trained from scratch on arithmetic. Curriculum + register scratchpads, stratified  evaluation. 
  - [claude_news](./claude_news) — up to the hour news about claude, sorted by personal interest
  - [BuzzOff](./BuzzOff) — Android alarm with vibration patterns
  - [livewall](./livewall) — HTML files as desktop wallpapers
  - [funnel](./funnel) — compress files for Claude (PDFs/images/Office → text)

  - [snappy](./snappy) — screenshots and GIF recorder
  - [pdftools](./pdftools) — split and merge PDFs (intended to combine with pdf2text)
